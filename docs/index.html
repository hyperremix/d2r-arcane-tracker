What is a Holy Grail in D2R?

The Holy Grail challenge in Diablo 2 Resurrected is a popular endgame goal where players aim to find every unique and set item in the game. D2R Arcane Tracker makes this challenge easier by automatically tracking your progress as you play.

Installation Guide

  1. 1
    Download the latest Windows installer (.exe) from GitHub Releases
  2. 2
    Run the installer - If Windows shows a "Windows protected your PC" warning, click "More info" then "Run anyway"
  3. 3
    Launch the application from the Start Menu
  4. 4
    Configure your D2R save file path (default: %USERPROFILE%\Saved Games\Diablo II Resurrected)
  5. 5
    Start tracking your Holy Grail progress automatically!
